# Disease Prediction Based on Symptoms (Machine Learning)

Project Overview:
-----------------

This project focuses on predicting diseases based on symptoms provided by the user using Machine Learning techniques.

Problem Statement:
-------------------

Users often experience symptoms but are unsure about the possible disease. This project aims to assist by predicting the most likely disease using ML models.

Approach:
---------

- Data cleaning and preprocessing
- Encoding target variables
- Feature preparation
- Training a classification model
- Predicting disease based on symptoms

 Technologies Used:
 ------------------
 
- Python
- Pandas, NumPy
- Scikit-learn
- Machine Learning (Classification)

 Output:
 -------
 
Given a set of symptoms as input, the model predicts the most probable disease.

Future Improvements:
--------------------
- Add NLP-based symptom understanding
- Use ensemble models
- Deploy using FastAPI
